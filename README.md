# 📌 Just Do It. - Animated To-Do List

A modern, responsive, and beautifully animated single-page To-Do List application. It features a clean, premium visual design with custom animations, built using HTML5, Vanilla JavaScript, CSS3, and Bootstrap 5.

---

## 🚀 Live Preview & Aesthetics

The application features:
- **Rich Typography & Aesthetics**: Configured with the modern `Poppins` font and custom hover states.
- **Dynamic Animations**: Includes an animated header loading sequence (`J-u-s-t do it.`) and smooth fade-in entry effects for new task items.
- **Modern Grid Layout**: A responsive task container displaying tasks in a clean grid that fits different device sizes.

---

## ✨ Features

- **Quick Add**: Input tasks and add them instantly by clicking **"I Got This!"** or pressing the `Enter` key.
- **Task Actions**:
  - 🟢 **Complete Task**: Toggle a strikethrough decoration on completed tasks.
  - 🔴 **Delete Task**: Remove tasks permanently from the list.
- **Fluid Interactions**: Hover micro-interactions on task items and action buttons.
